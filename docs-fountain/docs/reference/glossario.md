# Glossário

Este glossário define os termos técnicos e de negócio mais comuns usados na plataforma Fountain.

## Termos

| Termo | Definição |
|---|---|
| **Clawback** | Uma operação na XRPL que permite ao emissor de um token resgatar (queimar) tokens da carteira de um detentor. Usado no fluxo de burn. |
| **Colateral** | O ativo (XRP, RLUSD, etc.) depositado como garantia para a emissão de uma stablecoin. |
| **Emissor (Issuer)** | A entidade que cria e emite uma stablecoin. No contexto da Fountain, a Fountain é a emissora. |
| **Issued Currency** | Um token emitido em uma blockchain, como a XRPL, que é lastreado por um ativo externo. |
| **Mint** | O processo de criação de novos tokens de stablecoin. |
| **RLUSD** | Uma stablecoin lastreada em dólar emitida na XRPL. |
| **Stablecoin** | Um tipo de criptomoeda cujo valor é atrelado a um ativo externo, como uma moeda fiduciária (BRL, USD) ou uma commodity. |
| **Tokenizadora** | Uma empresa que transforma ativos do mundo real (como imóveis ou recebíveis) em tokens digitais em uma blockchain. |
| **Trustline** | Uma linha de confiança na XRPL que um usuário deve criar para poder receber um token emitido. |
| **Webhook** | Um mecanismo para notificar uma aplicação sobre eventos que ocorrem em outra aplicação, através de uma requisição HTTP. |
| **XRP** | O ativo nativo da XRP Ledger. |
| **XRPL (XRP Ledger)** | A blockchain descentralizada e de código aberto onde a Fountain emite suas stablecoins. |
